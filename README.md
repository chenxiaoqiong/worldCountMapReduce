# worldCountMapReduce
### 这是一个hadoop的mapReduce最简单的例子：###
#### 功能：统计输入文件中每个单词出现的次数 ####
#### 运行环境： ####
jdk 1.8
hadoop2.7.3

#### windows环境注意： ####
因hadoop是基于linux系统的，windows安装不能直接运行，会报各种错误，你需要安装hadooponwindows-master.zip这个包（安装方法，解压覆盖原来的hadoop目录即可）
